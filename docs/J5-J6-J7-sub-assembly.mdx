# Step 6: J5-J6-J7 Sub-Assembly

1. Install the `FL6700ZZ` bearing into the sheet metal component `J6_A`, and gently insert `J7_B` into the bearing, aligning it as shown in the figure.
<img src="/img/J5_J6_J7/prep_step1.png" alt="prep_step1" style={{ display: 'block', margin: '0 auto', width: '20%', maxWidth:'600px' }} />

2. Repeat the procedure with `J6_B`
<img src="/img/J5_J6_J7/prep_step2.png" alt="prep_step1" style={{ display: 'block', margin: '0 auto', width: '20%', maxWidth:'600px' }} />

3. Attach `J5_A` to the rotor of `J5` motor using 6 M3x18 bolts
<img src="/img/J5_J6_J7/step1.png" alt="step1" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

4. Attach the `J6_B` component to `J5_A` using 3 M3x8 bolts. Ensure that the `J6_B` is aligned towards the mechanical stop of `J5_A`
<img src="/img/J5_J6_J7/step2.png" alt="step2" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

5. Attach the component `J6_A` to `J5_A` using 3 M3x8 bolts.
<img src="/img/J5_J6_J7/step3.png" alt="step3" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

6. Attach component `J6_D` to motor `J6` using four M3x6 bolts. Ensure that `J6_D` is aligned so its top and bottom line up with the two Power+CAN ports.
<img src="/img/J5_J6_J7/step4.png" alt="step4" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

7. Attach `J7_E` to the rotor of `J6` using 6 M3x6 bolts
<img src="/img/J5_J6_J7/step5.png" alt="step5" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

8. Carefully place the motor `J6` between `J6_A` and `J6_B`, and securely fasten it to `J6_A` using 4 M3x6 bolts
<img src="/img/J5_J6_J7/step6.png" alt="step6" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

:::tip
When looking straight at `J6_A`, `J7_E` should be portruding to the left
:::

9. Attach part `J7_A` to the `J7` motor using three M3x8 bolts. Make sure the orientation matches the image, with the top bolt aligned with the Power+CAN port.
<img src="/img/J5_J6_J7/step7.png" alt="step7" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

10. Using two M3x8 bolts, mount part `J7_C` to the back of motor `J7` as shown. Check that itâ€™s aligned correctly and runs parallel to the line between the two Power+CAN ports.
<img src="/img/J5_J6_J7/step8.png" alt="step8" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

11. Insert bearings `MR106ZZ1` into the two holes of part `J7_D`.
<img src="/img/J5_J6_J7/step9.png" alt="step9" style={{ display: 'block', margin: '0 auto', width: '20%', maxWidth:'600px' }} />

12. Carefully place the motor `J7` between the `J6_A` and `J6_B` sheet metal pieces, and attach the two `J7_B` pieces to `J7_A` using 2 M3x8 bolts on each side.
<img src="/img/J5_J6_J7/step10.png" alt="step10" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

:::tip
`J7_C` and `J7_E` should face the same direction once this step is completed
:::

13. Attach the component `J7_D` to both `J7_C` and `J7_E` using the step bolt `DBSY4-6-5` with washers `SWSPS12-6-0.5` between the step bolt and `J7_D` as well as `J7_D` and `J7_C`/`J7_E`.
<img src="/img/J5_J6_J7/step11.png" alt="step11" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />


<h3><u>That concludes the J5-J6-J7 Sub-Assembly</u></h3>
