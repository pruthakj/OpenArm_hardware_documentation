# Step 7: End-Effector Sub-Assembly

1. Attach `J8_A` to `J8_B` using 3 M3x6 bolts
<img src={useBaseUrl('/img/end-effector/step1.png')} alt="step1" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

2. Attach `rail_connector` to the `J8` motor using 6 M3x8 bolts. Ensure that the flanges of `rail_connector` are aligned to the two Power+CAN ports of `J8`
<img src={useBaseUrl('/img/end-effector/step2.png')} alt="step2" style={{ display: 'block', margin: '0 auto', width: '50%', maxWidth:'600px' }} />

3. Carefully insert bearings `MR126ZZ` into `EE_linkage`. Note that two bearings need to be inserted from opposite directions.
<img src={useBaseUrl('/img/end-effector/step3.png')} alt="step3" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

4. Attach the two `EE_linkage` components to the `rotor_controller` using step bolts `DBSY4-5-4`. Make sure the bearings are oriented such that the fully exposed (open) side of each bearing faces toward the `rotor_controller`, as shown in the image. This orientation ensures proper alignment and freedom of movement for the linkages.
<img src={useBaseUrl('/img/end-effector/step4.png')} alt="step4" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

5. Connect `rotor_controller` to the rotor of `J8` using 4 M3x8 bolts.
<img src={useBaseUrl('/img/end-effector/step5.png')} alt="step5" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

6. Attach the linear guide to `rail_connector` using 4 M3x5 bolts.
<img src={useBaseUrl('/img/end-effector/step6.png')} alt="step6" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

7. Secure the `rail_CNC_attachment` to the slider blocks of the linear guide using four low-profile bolts `KBBS3-6`. Ensure that the `rail_CNC_attachment` extends in the same direction as the `EE_linkage`. This orientation simplifies the alignment and final assembly steps.
<img src={useBaseUrl('/img/end-effector/step7.png')} alt="step7" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

8. Attach the jaws to the `rail_CNC_attachment` using 4 low-profile bolts `KBBS3-6`. Ensure that the jaws are properly aligned for perfect assembly.
<img src={useBaseUrl('/img/end-effector/step8.png')} alt="step8" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />
<img src={useBaseUrl('/img/end-effector/step8_B.png')} alt="step8_B" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

9. Connect the two `EE_linkage` components to the `rail_CNC_attachment` using step bolts `DBSY4-5-4`. 
<img src={useBaseUrl('/img/end-effector/step9.png')} alt="step9" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

10. Mount the subassembly created in Step 1 to the rear face of the `J8` motor using four M3Ã—8 bolts.

- When assembling the right arm, ensure the `J8_A` part is aligned with the right jaw.

- When assembling the left arm, align `J8_A` with the left jaw.

The alignment ensures proper mechanical coupling and correct orientation of the jaws during operation, as indicated by the arrows in the image.
<img src={useBaseUrl('/img/end-effector/step10.png')} alt="step10" style={{ display: 'block', margin: '0 auto', width: '80%', maxWidth:'600px' }} />

<h3><u>That concludes the End-Effector Sub-Assembly</u></h3>
