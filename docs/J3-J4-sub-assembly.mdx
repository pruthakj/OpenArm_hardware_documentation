# Step 4: J3-J4 Sub-Assembly

1. Attach `J4_A` to the `J3` motor using 6 M3x6 bolts. Make sure the two Power+CAN ports face the front and rear for easy wiring.
<img src="/img/J3_J4/step1.png" alt="step1" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

2. Mount the bearing `FL6803ZZ` onto `J4_B`
<img src="/img/J3_J4/step2.png" alt="step2" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

3. Place the `J4_B` on the interior side of `J4_A`,aligning the holes. Do not fasten them yet. 
<img src="/img/J3_J4/step3.png" alt="step3" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

4. Carefully slide the `J4` motor between `J4_A` and `J4_B`. Align it with the holes on `J4_B`. Refer to the photo for correct positioning. Fasten the motor `J4` with part `J4_B` using 4 M3x6 bolts.
<img src="/img/J3_J4/step4.png" alt="step4" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

5. Fasten `J4_A` into `J4_B` using 6 M3x6 bolts
<img src="/img/J3_J4/step5.png" alt="step5" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

6. Finally, fasten `J4_A` to the motor `J4` using 6 M3x6 bolts.
<img src="/img/J3_J4/step6.png" alt="step6" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

7. Carefully insert the post `AETTS8-61-SC0-M3-N3` between the flanges of `J4_A` and fasten them with 2 M3x6 bolts
<img src="/img/J3_J4/step7.png" alt="step7" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />


<h3><u>That concludes the J3-J4 Sub-Assembly</u></h3>
