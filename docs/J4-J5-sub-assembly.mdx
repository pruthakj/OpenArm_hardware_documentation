# Step 5: J4-J5 Sub-Assembly

1. Attach `J4_C` to the rotor of `J4` motor using 6 M3x6 bolts
<img src="/img/J4_J5/step1.png" alt="step1" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

2. Mount the component `J4_D` onto the `FL6803ZZ` 
<img src="/img/J4_J5/step2.png" alt="step2" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

3. Attach the component `J4_E` to `J4_C` using 6 M3x6 bolts.
<img src="/img/J4_J5/step3.png" alt="step3" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

:::tip
Align the `J4_E` component vertically with the mechanical stop on `J4_C`, ensuring proper orientation before fastening.
<img src="/img/J4_J5/tip.png" alt="tip" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />
:::

4. Place the `J5` motor between the `J4_D` and `J4_E` and fasten them using 6 M3x6 bolts. Ensure that the Power+CAN ports are facing front and rear for effective wiring
<img src="/img/J4_J5/step4.png" alt="step4" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

5. Carefully place two posts `AETTS8-74.2-SC0-M3-N3` between the `J4_D` and `J4_E` components, and fasten them using M3x6 bolts
<img src="/img/J4_J5/step5.png" alt="step5" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

<h3><u>That concludes the J4-J5 Sub-Assembly</u></h3>
