# Final Steps

1. Attach `J2-J3 Sub-Assembly` to the rotor of `J3` motor using 6 M3x15 bolts
<img src={require('@site/static/img/final_assembly/step1.png').default} alt="step1" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

2. Attach `J2_A` to the rotor of `J2` motor using 6 M5x10 bolts. 
<img src={require('@site/static/img/final_assembly/step2.png').default} alt="step2" style={{ display: 'block', margin: '0 auto', width: '100%', maxWidth:'600px' }} />

3. Connect the `J8_A` component from the `End-Effector Sub-Assembly` to the rotor of `J7` motor using 6 M3x15 bolts.
<img src={require('@site/static/img/final_assembly/step3.png').default} alt="step3" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />

4. Finally, mount the arms on the pedestal using 4 M6x12 bolts in `J1_A`. Make sure you have already inserted T-slot nuts inside the aluminium extrusion. Align the top of `J1_A` to the top of the aluminium extrusion. 

:::tip
The Power+CAN ports of the `J1` motor of both the arms face the front. The reinforcement in the pedestal is towards the back.
:::
<img src={require('@site/static/img/final_assembly/step4.png').default} alt="step4" style={{ display: 'block', margin: '0 auto', width: '30%', maxWidth:'600px' }} />
