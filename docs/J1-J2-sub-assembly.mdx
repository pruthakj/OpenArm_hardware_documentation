# Step 2: J1-J2 Sub-Assembly

1. Attach the rear of the DM8009 motor to `J1_A` with 5 M4x10 bolts. Ensure the power and communication ports remain accessible. Ensure that this motor has been configured to have `ID:0x01` and `Master ID:0x11`
<img src={require('@site/static/img/J1_J2/step1.png').default} alt="step1" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

2. Attach `J1_B` to the motor using 3 M4x8 bolts. The mechanical stop should face the motorâ€™s power port and be on the top side. Refer to the image for clarity.
<img src={require('@site/static/img/J1_J2/step2.png').default} alt="step2" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

:::tip
For the left arm assembly, part `J1_B` must be oriented slightly differently. Please refer to the image for the correct orientation. All other steps remain unchanged. Ensure that the Power+CAN ports on the J1 motor face forward, with the mechanical stop positioned toward the ports and at the top.
<img src={require('@site/static/img/J1_J2/step2_left.png').default} alt="step2" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />
:::

3. Attach the `J1_C` part to the rotor of `J1` using 6 M5x10 bolts
<img src={require('@site/static/img/J1_J2/step3.png').default} alt="step4" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

4. Attach `J1_D` to `J1_C` using 2 M4x10 bolts
<img src={require('@site/static/img/J1_J2/step4.png').default} alt="step5" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

5. Attach `J1_E` to `J1_C` using 3 M4x10 bolts
<img src={require('@site/static/img/J1_J2/step5.png').default} alt="step6" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

6. Carefully insert the DM8009 motor `J2` between `J1_D` and `J1_E`, with the ports facing directly toward `J1`. The rear of `J2` should face `J1_E`. Join `J1_E` to `J2` motor using 2 M4x8 bolts
<img src={require('@site/static/img/J1_J2/step6.png').default} alt="step7" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

7. Attach `J1_D` to `J2` using 3 M4x8 bolts
<img src={require('@site/static/img/J1_J2/step7.png').default} alt="step8" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />
