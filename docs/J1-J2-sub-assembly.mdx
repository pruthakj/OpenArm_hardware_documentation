# Step 2: J1-J2 Sub-Assembly

1. Attach the rear of the DM8009 motor to `J1_A` with 5 M4x10 bolts. Ensure the power and communication ports remain accessible. Ensure that this motor has been configured to have `ID:0x01` and `Master ID:0x11`
<img src="/img/J1_J2/step1.png" alt="step1" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

2. Attach `J1_B` to the motor using 6 M4x8 bolts. The hole with 4 tapped holes should face the motorâ€™s power port and be on the top side. Refer to the image for clarity.
<img src="/img/J1_J2/step2.png" alt="step2" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

3. Attach `J1_C` to `J1_B` using 4 M3x6 bolts
<img src="/img/J1_J2/step3.png" alt="step3" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

4. Attach the `J1_D` part to the rotor of `J1` using 6 M5x10 bolts
<img src="/img/J1_J2/step4.png" alt="step4" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

5. Attach `J1_E` to `J1_D` using 2 M4x10 bolts
<img src="/img/J1_J2/step5.png" alt="step5" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

6. Attach `J1_F` to `J1_D` using 3 M4x10 bolts
<img src="/img/J1_J2/step6.png" alt="step6" style={{ display: 'block', margin: '0 auto', width: '40%', maxWidth:'600px' }} />

7. Carefully insert the DM8009 motor `J2` between `J1_E` and `J1_F`, with the ports facing directly toward `J1`. The rear of `J2` should face `J1_F`. Join `J1_F` to `J2` motor using 2 M4x6 bolts
<img src="/img/J1_J2/step7.png" alt="step7" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />

8. Attach `J1_E` to `J2` using 3 M4x6 bolts
<img src="/img/J1_J2/step8.png" alt="step8" style={{ display: 'block', margin: '0 auto', width: '60%', maxWidth:'600px' }} />
